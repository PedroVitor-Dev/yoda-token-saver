````md
# Yoda Token Saver

**Reduce tokens. Keep meaning.**

Yoda Token Saver is an open-source skill for creating shorter, clearer, and more efficient AI responses.

It helps AI assistants reduce token usage while preserving meaning, accuracy, and practical value.

The Yoda-inspired style gives the project a memorable identity, but the main goal is professional token optimization.

---

## Why this exists

AI responses are often longer than necessary.

This project helps reduce:

- Verbose explanations.
- Repeated context.
- Filler phrases.
- Overly long answers.
- Token waste.

Without losing:

- Meaning.
- Clarity.
- Accuracy.
- Important context.
- Useful instructions.

---

## Example

### Normal response

```txt
You should install the dependencies before running the application because the project requires external packages to work correctly.
````

### Optimized response

```txt
Install dependencies first. Then run the app.
```

### Yoda Mode

```txt
Dependencies install. Run after.
```

---

## Core Principle

```txt
Reduce tokens. Keep meaning.
```

Token saving should never make an answer confusing, unsafe, or incomplete.

---

## Modes

Yoda Token Saver supports multiple communication styles.

| Mode      | Purpose                                   |
| --------- | ----------------------------------------- |
| Default   | Short, professional, and clear            |
| Yoda      | Compact with light Yoda-inspired phrasing |
| Developer | Technical and direct                      |
| Military  | Step-by-step execution                    |
| Ultra     | Maximum token reduction                   |

## Installation

Copy the skill folder into your AI assistant skills directory:

```txt
skills/yoda-token-saver/
```

Then enable or reference the skill in your assistant workflow.

## Usage

Ask the assistant to use Yoda Token Saver.

Example:

```txt
Use Yoda Token Saver and explain this error.
```

Or request a specific mode:

```txt
Use Developer Mode.
```

