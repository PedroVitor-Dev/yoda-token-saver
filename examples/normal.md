# Normal Response Example

## User request

Explain how to fix a Git push rejected error.

## Normal AI response

This error usually happens when the remote repository contains commits that your local repository does not have yet. Git rejects the push to prevent you from accidentally overwriting changes made by someone else or from another machine.

To fix it, you need to pull the latest changes from the remote repository, merge or rebase them into your local branch, resolve any conflicts if they appear, and then push your changes again.
